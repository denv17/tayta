# Hero Block

Main landing section with title, description, CTA button, and image.

## Files

- `hero.twig` - Template file
- `fields.json` - ACF field group definition

## Fields

- **icon** (image) - Icon or logo image
- **title** (text) - Main heading
- **description** (textarea) - Description text
- **cta** (link) - CTA button with URL, title, and target
- **image** (image) - Main hero image (bowl)
